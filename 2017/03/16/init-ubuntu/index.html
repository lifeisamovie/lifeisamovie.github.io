我在Linux上使用的一些工具

发表于 2017-03-16

前几天学习Linux，本着“Learn It and Use It for Life”的原则装了Ubuntu。换了新的环境，首先就是找一些替代软件保证日常使用学习正常进行，都是一些基础的软件，在这里做个总结，以备日后之需。

1.浏览器
一直使用Google Chrome，已经习惯了，基本上是使用最多的软件了，内存有一半被它占去了…同步功能很好用，书签、插件都可以，省了很多力气。
2.VMware Workstation
虽然使用着Linux，但是还会需要装一些其它发行版做测试。快照功能很好用。
3.下载
就我目前的使用体验来看，迅雷确实强大，Linux平台上基本上没有可与之一战的软件。不过配置好的话，Linux下也可以有一个方便的下载环境。我使用的是wget/axel+Aria2+Transmission，wget下一般文件够用了，Aria2搭配一些Chrome插件用来下载百度盘的东西，Transmission用来下种子文件。
Aria2配置稍麻烦些，我使用的是这个BaiduExporter，里面有用法，搭配这个网址Aria2 WebUI，秒变神器！
4.桌面配置
虽然有人批评桌面美化纯粹是浪费时间的事情，但是我真的认为一个漂亮的桌面能让你有一个十分愉悦的使用体验:)，而且也不花多少时间。我是先装了Unity Tweak Tool桌面管理软件，然后使用了Flatabulous主题，最终效果如下。

5.编辑器
作为一个日后要从事运维/安全的人来说，Vim当然是不二之选，因为它可以在命令行下使用。但我并不是什么Vim党，不管什么编辑器，你只要用好一个提高效率就行。但是，虽然我还是个Vim初级用户，它真的挺好用的！这篇简明练级攻略很好，马上就能上手。具体配置可以等到日后需要在折腾。
6.shadowsoks
必备，现在VPS可以说是最好的FQ方案了吧，其它不多说了。
7.其它

  • 办公：WPS，卸载了自带的office软件;PDF用Foxit Reader
  • 电子邮件：自带的Thunderbird就行
  • Redshift：一个护眼软件，我的必备
  • 影音播放：网易云音乐，SMPlayer，VLC等
  • 快捷键：最常用的是截图，在Win上使用的是FastStone，这里去系统设置配置一下快捷键就行
  • Shell脚本：其实Linux本身就有很多高效的工具，也可以自己写一些工具

配置工具是一个一劳永逸的事情，可以大大提高效率、节省时间。最后，谈谈我短暂使用后的几点感想吧。

  1. 使用Linux真的很爽！因为它让你有一种掌控感，你了解它，使用它会让你觉得自己有超能力，而Win就从来没有:(
  2. OS就是一个工具，挑适合的、能提高你工作学习效率的用就行。对于我，工作学习我会毫不犹豫的选Linux，而娱乐的话，当然是Win了。
