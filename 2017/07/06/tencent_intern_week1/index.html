腾讯实习日志，第一周，7.5-7.8

2017.07.05, 周三

  今天上午在同学的帮忙下总算是顺利的入职了，在这里要对他说声谢谢。入职之后已经接近中午，导师也正好不再，所以他就安排组里的一位同事来接待我。从腾大坐班车到朗科，放下行李后就跟着他吃中饭去了（额~那个窗口卖的饭确实有点难吃…）。
  吃完中饭后，就开始装电脑、入域什么的，弄好以后已经两点多了，不久导师也回来了，但是因为上不了内网，也干不了什么，所以我就下载一些软件，准备以下环境。
  下午，好像是周会，有点儿懵逼，好像是做的PPT被老板批评了，不清楚，但是在会上看见了一面面试官，只是他不再留长头发了…
  晚上导师带着我还有小组里的其它几位同事去外面吃饭，顺便认识了一下。
  额，总之，第一天什么也没干，就是准备一下环境、认识了几位同事。

2017.07.06, 周四

  上午还是继续装软件，准备环境；然后，导师带着我认识了组里其它同事，虽然早对着rtx看了一遍，但还是没记住这么多人，嗯，慢慢记吧。
  然后导师给我大体安排了一下实习期任务，根据任务需要学习的以下几个知识点：

  • LVS（TGW）
  • Python Web（用来做界面）：Flask/Django, Bootstrap
  • 复习网络知识：路由协议（RIP/OSPF/BGP）等

  下午五点左右和导师还有另一位前辈开了一个简短的沟通会，主要是就实习期做了大致安排、信息交流和一些建议。只是提出两个大的方向，一是TGW相关的，二是平台建设相关，具体任务还要进一步明确。此外，还叮嘱了一些注意事项，如信息安全。
  前辈给的一些建议也是十分中肯的，

  • 沟通：不仅仅是组内沟通，也要与其它组沟通，开拓视野，增长知识，提高技能；
  • 明确实习对自己的意义：实习期间要学习什么，达到一个怎样的效果？这是要思考的。

  接下来短期内要做的：

  1. 学习上面那些知识；
  2. 继续认识同事；
  3. 了解部门的组织结构、每个分小组的工作内容

  自己还有一些想知道的其它问题，如是否有转正机会（人头hc）等等，但现在来说这些还不是那么重要，所以暂且搁置。当下最重要的还是：尽快熟悉业务，明确实习目标和任务，融入环境。
