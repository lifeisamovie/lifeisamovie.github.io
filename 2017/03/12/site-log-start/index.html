小站做好了～

发表于 2017-03-12 | 分类于 小站日志 |

终于把这个博客弄好了，老早就想弄一下，过去也折腾过Jekyll，可是又没什么可写的（渣渣…)，读书笔记写到WizNote里面就够了，于是也就不了了之了。可是这两天在网上闲逛，逛到一个个人小站iTimothy，里面记了博主日常点滴与技术分享，感觉还是蛮有趣的，浩瀚的互联网的某个角落里藏着你的故事，amazing….

独立域名和服务器就暂时不搞了，暂时挂在Github上。这回选用了Hexo，真是超级简单，Github上找了一个中意的主题nexT(star数真多！），几个小时就搞定了，效果也挺令人满意的，简洁又美观，哈哈。

由于在Github上设置了强制使用https，所以无法访问的时候，在网址前面加上https。
