小站做好了～

发表于 2017-03-15 | 分类于 小站日志

终于把这个博客弄好了，老早就想弄一下，过去也折腾过Jekyll，可是想想又没什么可写的（水平有待提高…)，读书笔记写到WizNote里面就够了，于是也就不了了之了。可是这两天在网上闲逛，逛到一个个人小站iTimothy，里面记了博主日常生活与技术分享，感觉还是挺有趣的，想想互联网的某个角落里藏着你的点滴，也是蛮神奇的^^

独立域名和服务器就暂时不搞了，暂时挂在Github上。这回选用了Hexo，真是超级简单，Github上找了一个中意的主题nexT(star数真多！），几个小时就搞定了，效果也挺令人满意的，简洁又美观，哈哈。

由于在Github上设置了强制使用https，不知道为什么在主页上点分类或标签这些链接时会转换成http链接，导致无法访问！而且现在也无法设置使用http链接，除非重新创建整个仓库，这里有个issue。所以无法访问的时候，在地址栏前加上https；或者让你的浏览器强制对某个网站使用https，谷歌浏览器的方法在这里（推荐，这个其实挺有用的）。
